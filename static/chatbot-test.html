🤖 PawfectBot - Floating FAQ Chatbot

A lightweight, self-contained chatbot widget for Pawfect Finds

✨ Features

  • Floating chat button (bottom-right corner)
  • Smooth open/close animations
  • 5 pre-configured FAQ questions with instant responses
  • Beautiful message bubbles (user green, bot gray)
  • Responsive design for all devices
  • Zero external dependencies (pure vanilla JS)
  • No backend required
  • Instant deployment

  1. CSS Import - Added to <head> in base.html
  2. JavaScript Import - Added before </html> in base.html
  3. Auto-initialization - Chatbot loads automatically on all pages
